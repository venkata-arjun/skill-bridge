# Skill Bridge

![Homepage](assets/hero-image.png)

## Overview

Skill Bridge is a web platform designed to connect students, faculty, and speakers for knowledge-sharing sessions. The website streamlines the process of proposing, approving, and attending educational sessions, making it easy for users to participate in and organize learning events.

## Purpose

The purpose of Skill Bridge is to:

- Facilitate the discovery and proposal of new topics for sessions.
- Enable students to request sessions and become speakers.
- Provide faculty with tools to approve or reject session proposals.
- Allow speakers to manage their sessions and receive feedback.

## Features

- **User Authentication:** Secure signup and login for students, faculty, and speakers.
- **Session Proposals:** Students can propose new topics and request sessions.
- **Speaker Management:** Users can apply to become speakers and manage their sessions.
- **Faculty Approval:** Faculty can review, approve, or reject session proposals.
- **Feedback System:** Attendees can provide feedback and ratings for sessions.
- **Dashboards:** Personalized dashboards for students, speakers, and faculty to manage their activities.
- **Session Listings:** Browse upcoming and past sessions with detailed information.
